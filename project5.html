    def dropOff (carrying):
    if(carrying==False):

        dropLabel1.value='Drop off'

        dropLabel1.toBack()
        dropLabel2.toBack()

    #here is an instance of it being false#
    if(packageGroup1.visible==True):
        if(packageGroup1.hits(truck.centerX,truck.centerY)==True):
            packageGroup1.visible=False
            dropOff(True)

    #here is an instance of it being true#
    if(dropOffRect.contains(truck.centerX,truck.centerY)==True):
        dropOff(False)

This code determines whether or not a package can be dropped off. If a package is picked up, because 'dropOff' is then marked as "True", the collection zone will turn green and will allow for the package to be dropped off. After it is dropped off, it is marked as "False", which reverts it back to its red state in which a package cannot be dropped off. This allows for the game's core objective to function, as if you couldn't drop it off or could drop it off without a package, the game would break.
