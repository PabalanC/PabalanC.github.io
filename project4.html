Project 4: Environmental Consciousness Quiz

Description:In this project, we were tasked with using key functions, contains, hits, as well as previously used if statements and object properties. This quiz will give you a score based on your choices, in which a lower score is desired. Each question has one good answer, which earns no points, two okay answers, which earn one point, and one bad answer, which earns two points. Once you answer all six questions using the number keys (1, 2, 3, and 4), you can reveal your score at the end, 0 being the best and 12 being the worst.

Concepts Learned:The important concepts that we learned were how to use contains, hits, and key functions. The key functions were used in advancing the questions (pressing '1', '2', '3', or '4'), and are important because they allow for more versatility in program creation when working alongside mouse functions. The 'hits' and 'contains' attributes were used in the start button and final score. These are important since they easily allow for button UI to appear more flashy and intuitive, by having them change color when hovering over them, or when pressing them specifically. These were used in the start button, and the circle at the end to reveal your score
