Project 2: Global Scholars Goal 2 Slideshow

Description: This project is a slideshow that walks through the details of goal 2 of the Global Scholars program, being the zero hunger mission. It walks through the background information for the goal, the targets of the goal, and things that people can do to help.

Concepts Learned: Through this project, I learned how to effectively use functions, such as mouse events. I also learned how to manipulate shape properties and use variables.

Meaningful Code Snippet:

    #this is CJ's logo
def cjLogo(cjX,cjY,cjColor1,cjColor2):

    Line(cjX-25,cjY-20,cjX-25,cjY-45,fill=cjColor2,lineWidth=4, rotateAngle=-15)
    Line(cjX,cjY-25,cjX,cjY-50,fill=cjColor2,lineWidth=4)
    Line(cjX+25,cjY-20,cjX+25,cjY-45,fill=cjColor2,lineWidth=4, rotateAngle=+15)

def akLogo(akX,akY,akColor1,akColor2):
    Circle(akX,akY,35,fill=akColor1)
    Rect(akX-40,akY-35,75,30,fill=akColor2)
    Line(akX,akY,akX,akY-40,fill=akColor1)
    Line(akX-15,akY,akX-15,akY-40,fill=akColor1)
    Line(akX+15,akY,akX+15,akY-40,fill=akColor1)

These two functions are the graphics found on each slide of the presentation. This makes it so the two shapes' locations and colors can be manipulated easily.
